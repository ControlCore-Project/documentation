The CONTROL-CORE Platform
====================================================================================

**CONTROL-CORE** is a framework for Closed-Loop peripheral neuromodulation control systems

Check out the :doc:`installation` section for further information on installing the CONTROL-CORE framework. 
Check out the :doc:`usage` section for further information on writing ``concore`` programs.


.. note::

   This project is under active development.
